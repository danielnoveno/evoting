# VoteChain MVP Workspace

Workspace awal untuk pengerjaan MVP skripsi VoteChain (14 hari).

## Folder Utama
- `contracts/` — smart contract + test Foundry
- `frontend/` — Next.js app minimum
- `shared/` — ABI, address, konstanta jaringan
- `evidence/` — artefak dan bukti BAB IV/V
- `.opencode/handoff/` — koordinasi antar-agent (wajib di dalam .opencode)
- `indexer/` — placeholder P1

## Prinsip kerja
1. Keamanan inti dulu (phase, whitelist, commit-reveal, anti double action).
2. Baru integrasi UI minimum.
3. Semua klaim skripsi harus ditautkan ke bukti.
